# template-ng-tailwind-jit

- **template of:** angular cli
- **tailwindcss:** `^2.2.19`

## Started
with npm:
```bash
npm install
```
with yarn:
```bash
yarn
```

---
## Develop
with npm:
```bash
npm start
```
with yarn:
```bash
yarn start
```

---
## Build
with npm:
```bash
npm run build
```
with yarn:
```bash
yarn build
```
